# Hyprland Lua Configuration Setup

## Overview

As of Hyprland 0.55+, configuration has been migrated from `.conf` (hyprlang) to **Lua** format. The setup is now modular across multiple `.lua` files.

## File Structure

```
.config/hypr/
├── hyprland.lua        # Main entry point, loads all modules
├── look_and_feel.lua   # Colors, style, general, decoration, animations, xwayland, env vars
├── monitor.lua         # Monitor configuration
├── input.lua           # Keyboard/mouse input settings
├── keybinds.lua        # All keybindings and dispatcher functions
├── windowrules.lua     # Window rules (float, size, workspace, opacity, etc.)
├── autostart.lua       # Autostart applications and services
├── drawers.lua         # Auto-generated special workspace drawers (gen-drawers.sh)
├── colors.conf         # Color definitions (legacy format, sourced by look_and_feel.lua)
├── style.conf          # Style variables (legacy format, sourced by look_and_feel.lua)
└── scripts/            # Helper scripts (gen-drawers.sh, toggle-record.sh, etc.)
```

## Key Changes from Legacy .conf

| Legacy (.conf) | Current (.lua) |
|----------------|----------------|
| Single `hyprland.conf` with `source = ...` | Modular `require()` in `hyprland.lua` |
| `gaps_in = $gaps_inner` | `gaps_in = style.gaps_inner` (from style table) |
| `monitor = ,2560x1440@144,0x0,1.6` | `hl.monitor({output="", mode="preferred", position="auto", scale=1.6})` |
| `env = PATH,$PATH` | Removed (PATH handled by system) |
| `env = XCURSOR_THEME,$cursor_theme` | Hardcoded: `Bibata-Modern-Classic`, size 24 |
| XWayland in main config | `hl.config({xwayland = {force_zero_scaling = true}})` in `look_and_feel.lua` |
| `windowrule = ...` syntax | `hl.window_rule({match = {...}, ...})` |
| `bind = KEY, dispatcher` | `bind("KEY", hl.dsp.dispatcher(...))` |

## Module Details

### `hyprland.lua` — Main Entry Point
```lua
require("look_and_feel")
require("monitor")
require("input")
require("keybinds")
require("windowrules")
require("autostart")

-- Load drawers if generated (auto-generated by gen-drawers.sh on startup)
local drawers_path = os.getenv("HOME") .. "/.config/hypr/drawers.lua"
local f = io.open(drawers_path, "r")
if f then f:close(); require("drawers") end

hl.config({ autogenerated = false })
```

### `look_and_feel.lua` — Visuals & Environment
- **Colors table**: background, foreground, accent, borders, shadows
- **Style table**: border_size, gaps_inner/outer, corner_radius, fonts, cursor, shadows
- **Environment variables**: XCURSOR_THEME, XCURSOR_SIZE, XDG_*, ELECTRON_OZONE_PLATFORM_HINT, WLR_DRM_NO_MODIFIERS
- **General**: gaps, borders, colors, layout
- **Decoration**: rounding, opacity, blur (disabled), shadows
- **Animations**: linear/snappy curves, windows, fade, workspaces
- **XWayland**: `force_zero_scaling = true`

### `monitor.lua` — Display
```lua
hl.monitor({
    output = "",
    mode = "preferred",
    position = "auto",
    scale = 1.6
})
```

### `input.lua` — Keyboard/Mouse
```lua
hl.config({
    input = {
        kb_layout = "pt",
        repeat_rate = 25,
        repeat_delay = 300,
        numlock_by_default = true,
        accel_profile = "flat",
        sensitivity = 0
    }
})
```

### `keybinds.lua` — Keybindings
- **Dispatcher helpers**: `bind(keys, dispatcher, flags)`, `workspace(arg)` for Waybar compatibility
- **Apps**: terminal (ghostty), launcher (walker), browser (zen-browser), steam, heroic, antigravity, antigravity-ide, spotify, nautilus, hyprlock
- **Window management**: close, fullscreen, float, pseudo, focus, move, resize
- **Workspaces**: SUPER+1..10, SUPER+SHIFT+1..10, keypad, SUPER+ALT+left/right
- **Screenshots**: Print, SHIFT+Print, SUPER+SHIFT+S (grim + slurp + wl-copy)
- **Recording**: SUPER+SHIFT+R (toggle-record.sh)
- **Media/Volume/Brightness**: XF86 keys with playerctl, wpctl, brightnessctl, swayosd-client
- **Mouse**: SUPER+mouse drag/resize
- **Theme toggle**: SUPER+SHIFT+T

### `windowrules.lua` — Window Rules
- **Float rules**: portals, file dialogs, Bitwarden, pavucontrol, Nautilus, imv, mpv, btop, nmtui, bluetui
- **Size rules**: Nautilus (900x600), pavucontrol (800x600), Bitwarden (1000x700), btop (1200x800)
- **Workspace rules**: zen→1, ghostty→2, vesktop→3, antigravity→4, antigravity-ide→5, Hermes→6, Spotify→9, steam/heroic/Overwatch→10
- **Special workspace drawers**: btop, nmtui, bluetui, pavucontrol (defined in drawers.lua, matched here)
- **Opacity**: ghostty=1.0, antigravity/antigravity-ide/zen=0.92, vesktop=0.95, Bitwarden=1.0
- **Suppress maximize**: steam, heroic, vesktop
- **No initial focus**: vesktop, steam

### `autostart.lua` — Session Startup
- **Core services**: awww-daemon, waybar, swaync, elephant, walker, polkit, wl-paste/cliphist, hyprsunset
- **DBus environment**: WAYLAND_DISPLAY, XDG_*, HYPRLAND_INSTANCE_SIGNATURE
- **Wallpaper**: awww with wipe transition
- **Drawers generation**: gen-drawers.sh + hyprctl reload
- **Keyring**: gnome-keyring-daemon
- **Apps**: vesktop, spotify, steam (-silent), heroic (--no-gui), bitwarden
- **xdg-desktop-portal**: restart in Hyprland session context

### `drawers.lua` — Auto-generated Special Workspaces
Generated by `gen-drawers.sh` on startup:
- `special:btop` → ghostty -e btop (1400x600, pos 100,39)
- `special:nmtui` → ghostty -e nmtui (700x420, pos 450,39)
- `special:bluetui` → ghostty -e bluetui (700x520, pos 450,39)
- `special:pavucontrol` → pavucontrol (800x600, pos 400,39)

## Removed Legacy Files
The following `.conf` files are **no longer used** and should be removed:
- `hyprland.conf` (replaced by `hyprland.lua` + modules)
- `keybinds.conf` (replaced by `keybinds.lua`)
- `windowrules.conf` (replaced by `windowrules.lua`)
- `autostart.conf` (replaced by `autostart.lua`)
- `drawers.conf` (replaced by `drawers.lua`)
- `colors.conf` / `style.conf` kept as data sources for `look_and_feel.lua`
